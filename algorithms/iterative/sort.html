Iterative Sort

Iterative sorting algorithms sort a collection using loops rather than recursion. All three algorithms below sort in ascending order.

  1. Bubble Sort

    2. Repeatedly compare adjacent elements and swap them if out of order. Time complexity: O(n²).
  2. Selection Sort

    3. Repeatedly find the minimum element in the unsorted portion and move it to the front. Time complexity: O(n²).
  3. Insertion Sort

    4. Build the sorted array one element at a time by inserting each element into its correct position. Time complexity: O(n²) worst case, O(n) best case.
